William S Fuller Resume API

About

While working at Postman one thing that was super imporant was the power of APIs. I wanted to put together a fun project mainly focused on doing some back end work with NodeJS and MongoDB. What I came up with was the idea of turning my resume into an API.

Contained within this API are 4 endpoints: 

/api/personal
/api/technology
/api/experience
/api/education

  • Personal
  • Technology
  • Experience
  • Education
